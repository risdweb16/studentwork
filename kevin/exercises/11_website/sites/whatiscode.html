WHAT IS CODE?

One of the more intriguing topics within the article showcases the community driven aspects of developing code. From the section titled And Now for Something Beautiful, Ford discusses the way in which GitHub allows for a world-wide community to evolve codes published onto the site, all in a very optimistic approach and manner. Writing code and publishing them on servers like GitHub is a collaborative effort to develop a file in a way that eliminates the boundaries that is often seen in other fields. Whereas in architecture there is community-engaged design and many firms rely on collaboration, there are limitations to this that GitHub overcomes. In what ways can the database and community driven development of GitHub be translated into other fields in order to continually progress?
